# PDF Library — фінальна структура

## Як користуватись

1. Поклади PDF-книги в папку `books/`.
2. Запусти локально:

```bash
python library_builder.py
python -m http.server 8000
```

3. Відкрий:

```text
http://localhost:8000
```

## GitHub Pages

1. Завантаж усі файли в корінь репозиторію.
2. Увімкни `Settings → Pages → Source → GitHub Actions`.
3. Після push workflow сам згенерує `index.html` і опублікує сайт.

## Reader

- Кожна PDF-сторінка вважається розворотом.
- Порядок читання: ліва половина → права половина → наступний розворот.
- На мобільному є велика кнопка `НА ВЕСЬ ЕКРАН`.
- Працюють свайпи, тап по краях, zoom `+ / −`.
- Назви PDF кирилицею підтримуються.
