# PDF Library — fullscreen rework

Це повний GitHub Pages проєкт для PDF-бібліотеки.

## Що виправлено

- Повноекранний reader більше не показує білий прямокутник-контейнер поверх сторінки.
- Canvas PDF займає повну ширину екрана.
- Панелі керування накладаються зверху/знизу і можуть ховатися тапом по центру.
- На мобільному читання йде: ліва половина PDF-сторінки → права половина → наступна PDF-сторінка.
- Є 4 теми: світла, темна, мʼятна, тепла.
- Працюють свайпи, тап по краях, zoom +/−, збереження прогресу.

## Як користуватись

1. Поклади PDF у `books/`.
2. Локально:

```bash
python library_builder.py
python -m http.server 8000
```

3. GitHub Pages:

```text
Settings → Pages → Source → GitHub Actions
```

4. Зроби push і дочекайся зеленого статусу Actions.

## Важливо

Якщо на телефоні бачиш стару версію, відкрий сайт з параметром `?v=fullscreen2` або в приватному режимі.
