# Stitch Reader FORCE UPDATE v3

Ця версія має явний жовтий маркер `ОНОВЛЕНО: FORCE UPDATE v3`. Якщо маркер не видно — GitHub Pages або браузер показує старий `index.html`.

## Оновлення
1. Видалити старі файли в репозиторії, крім `books/` з PDF.
2. Завантажити всі файли з цього архіву в корінь репозиторію.
3. Зробити commit/push.
4. Перевірити зелений статус Actions.
5. Відкрити сайт з `?v=force-update-v3`.
