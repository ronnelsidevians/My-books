# PDF Library mobile v3

Оновлення: мобільний full-screen reader, свайпи, тап по краях, приховування панелей, чіткіший текст через devicePixelRatio, розділення PDF-розвороту на ліву/праву половину на мобільному, повільне перегортання.

## Використання
1. Розпакуй архів у корінь репозиторію.
2. Додай PDF у `books/`.
3. Зроби commit/push.
4. GitHub Pages: `Settings → Pages → Source → GitHub Actions`.
5. Дочекайся зеленого статусу в `Actions`.
