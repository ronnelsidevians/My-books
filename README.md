# PDF Library mobile v5

- Кнопка fullscreen `⛶` у reader.
- Перегортання суворо почергове: ліва половина розвороту → права половина розвороту → наступний розворот.
- Половина розвороту рендериться на всю ширину екрана.
- Якщо сторінка висока, вона скролиться вертикально.
- Свайпи вліво/вправо та тап по краях працюють.
- `+`/`−` змінюють масштаб тексту.

На iPhone Safari системний fullscreen і примусовий landscape можуть бути обмежені браузером, але reader все одно перекриває весь viewport сторінки.
