# PDF Library mobile v4

Reader v4:
- reader займає 100vw/100dvh;
- на мобільному сторінка рендериться на повну ширину екрана, а висота скролиться;
- текст чіткіший: devicePixelRatio + render boost;
- є zoom +/-;
- кнопка 1/2 перемикає половину розвороту / повний розворот;
- свайпи, тап по краях, повільне перегортання;
- кнопка fullscreen пробує увімкнути fullscreen і landscape.

На iPhone Safari повноекранний режим і примусова орієнтація можуть блокуватися браузером, але reader все одно перекриває весь сайт.
